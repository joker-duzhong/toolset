# 📝 【在问高考】项目全景开发记录与 TODO LIST (V1.0)

## 💻 前端工程与交互体验 (Frontend)
**体验目标**：极简美学、移动端优先、全链路零刷新 (Zero-Reload)、微动效与情绪安抚。

### ⚙️ 模块一：全局规范与网络基建 (Infra)
- [ ] **技术栈与 UI 规范**：React + Tailwind CSS + Framer-motion。全面采用 Bento Box (便当盒) 卡片布局，广泛使用毛玻璃 (`backdrop-blur`) 与浅色边框。
- [ ] **零刷新状态管理 (Zero-Reload UX)**：
  - 全局接入 `@tanstack/react-query` 或 `SWR`。
  - 绝对禁止使用 `window.location.reload()` 阻断沉浸感。
- [ ] **网络拦截与容错**：
  - 统一配置 Vite 代理解决开发跨域。
  - 配置 Axios 全局拦截器，遇到 429 (限流) 或 500 (崩溃) 时，弹出统一的 Toast 提示（如：“引力波干扰，请重试”），确保页面不白屏。

### 🌳 模块二：双面树洞 UI & 交互
- [ ] **Tab 切换容器**：使用 framer-motion `layoutId` 实现 Emo区(暖色) 与 求助区(冷色) 切换时，底部游标的平滑滑动。
- [ ] **发帖防抖与无感插入**：
  - 移动端使用底部 Drawer，PC 使用 Modal 承载发帖表单。
  - 发送成功后关闭弹窗，利用 `mutate/invalidateQueries` 重新拉取数据，新卡片带有**向下推挤+淡入**的动效出现。
- [ ] **无限滚动信息流 (Infinite Scroll)**：
  - 封装 `IntersectionObserver` 监听底部哨兵元素。
  - 结合后端的 Cursor 分页实现无缝下拉加载，底部配有“到达树洞最深处”提示。
- [ ] **极速交互与 AI 轮询**：
  - **Optimistic UI (乐观更新)**：点击“抱抱”按钮瞬间，执行回弹缩放动效且数字 +1，不等待接口返回。
  - **AI 气泡轮询**：对于自己刚发的且没有 AI 回复的帖子，前端启动 `{ refreshInterval: 3000 }` 的短轮询，展示流光骨架屏。一旦拿到数据，用微动效展开 AI 专属气泡。

### ⚖️ 模块三：红黑榜 UI & 交互
- [ ] **结构化发布表单**：舍弃大文本，改为极简的“填空题/档案卡”样式 Input。
- [ ] **对抗态势卡片 (信息流)**：
  - 包含极简搜索栏。
  - 卡片底部固定两个毛玻璃按钮：🔴 踩坑预警 vs 🟢 逻辑通过。
- [ ] **深度质询视图 (详情页)**：
  - **顶端抗衡条**：一条红绿两端延伸的进度条（依票数动态拉伸）。
  - **AI 报告区**：带有轻微发光边框 (Glow Effect) 的 AI 总结卡片置顶展示。
  - **胶囊弹幕流**：底部的用户短评分为左右两列，采用圆角 Pill/Tag 样式展示。

### 🥷 模块四：个人中心与设置 UI
- [ ] **Bento 马甲摇号器**：
  - CSS Grid 错落布局。点击 🎲 按钮，执行 3D 翻转动效并平滑更新马甲名称和头像。
  - 状态选择器：横向平滑滚动的 Emoji 列表。
- [ ] **数值增长动效**：使用 `useSpring` 让获赞数等统计指标从 0 滚动到真实数字。
- [ ] **我的夏日存档交互**：
  - 历史列表支持左滑 (Swipe-to-left) 呼出红色的【🗑️ 焚毁】按钮。点击后卡片缩小消失 (`exit={{ scale: 0.8, opacity: 0 }}`)。
- [ ] **危险操作与防误触 (Wipe Modal)**：
  - 点击【抹除痕迹】，弹出红色警戒风 Modal。强制倒数 5 秒后，确认按钮才解除 Disable 状态。
- [ ] **合规与法律文案抽屉 (Drawers)**：
  - 底部放置三个暗色链接。点击后从底部优雅升起 Drawer（无需跳出页面），展示《社区生存法则》、《志愿 AI 免责声明》、《🆘 紧急援助专线》。
